# Font Files

## Magnetik Font
To use the Magnetik font, you need to add the following font files to this directory:

- `Magnetik-Medium.woff2` (500 weight)
- `Magnetik-Medium.woff` (500 weight fallback)
- `Magnetik-Bold.woff2` (700 weight)
- `Magnetik-Bold.woff` (700 weight fallback)

You can obtain these files from:
1. The font foundry/designer
2. Font licensing platforms
3. Your design team

## Google Fonts
Space Grotesk is loaded from Google Fonts and doesn't require local files.

## Usage in Components
```jsx
// Space Grotesk
className="font-grotesk font-medium" // 500 weight
className="font-grotesk font-bold"   // 700 weight

// Magnetik
className="font-magnetik font-medium" // 500 weight
className="font-magnetik font-bold"   // 700 weight
```
